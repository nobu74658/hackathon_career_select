# ハッカソン経験談 LT資料

5分間のハッカソン経験談プレゼンテーション資料です。

## 内容

- ハッカソン参加歴（優勝1回、準優勝2回、企業賞1回）
- 成長ストーリー（Flutter独学からコミュニティ運営まで）
- ハッカソンから得た5つの学び
- Engineer Guild Hackathon「あみぬり」アプリの紹介

## スライド実行方法

### 開発サーバー起動
```bash
npm run dev
```

ブラウザで http://localhost:3030 にアクセス

### PDF出力
```bash
npm run export
```

### ビルド（静的サイト）
```bash
npm run build
```
